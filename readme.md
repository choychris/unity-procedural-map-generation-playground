### About

Prototype playground for procedural generation of Don't Starve-like world map, using Voronoi diagram.

#### Library used

- [unit-delaunay](https://github.com/OskarSigvardsson/unity-delaunay)

#### Reference

- [Sebastian Lague, Procedural Terrain Generation](https://github.com/SebLague/Procedural-Landmass-Generation/tree/master)
- [redblobgames, Polygonal Map Generation for Games](http://www-cs-students.stanford.edu/~amitp/game-programming/polygon-map-generation/)
- [Iain McManus, Unity Tutorial: Procgen Terrain Meshes](https://www.youtube.com/watch?v=QaE3RPlNMgE&list=PLkBiJgxNbuOW6pgxOkYZhTsCOgvTsEBgk&index=13)
